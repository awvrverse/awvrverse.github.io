# Blog Portal Modern - Jekyll + Material Design 3

Blog modern dengan fitur lengkap menggunakan Jekyll dan Material Design 3.

## Fitur

- ✅ Material Design 3
- ✅ Responsive Design
- ✅ Dark/Light Mode
- ✅ Infinite Scroll
- ✅ SEO Optimized
- ✅ PWA Support
- ✅ Syntax Highlighting
- ✅ Search Functionality
- ✅ Markdown Support

## Instalasi

1. Clone repository ini
2. Install dependencies:
```bash
   bundle install
```
3. Jalankan server development:
```bash
   bundle exec jekyll serve
```
4. Buka browser di `http://localhost:4000`

## Deploy ke GitHub Pages

1. Push ke repository GitHub
2. Aktifkan GitHub Pages di Settings
3. Pilih branch `main` dan folder `/ (root)`
